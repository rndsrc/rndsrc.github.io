New Jupyter-Book Website
2019/03/29

After using Wordpress and an NAS to host my personal website for 5 years, I finally decided to switch. The main reason is that I evolved into a total git-geek---I want to version control absolutely everything in git. Wordpress' database content management is overkill for me. Instead, all I need is a static website versioned by git and hosted by GitHub.

In addition, as I use Jupyter Notebook/Lab extensively to process EHT data, I would love to integrate Jupyter Notebooks into my website. The Jupyter-Book project seems to be the perfect solution. It supports both markdown (like jekyll) and Jupyter Notebooks for content creation, and also support using Binder to run notebooks live.
