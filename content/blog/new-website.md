# A New Jupyter-Book Based Website

After using Wordpress and an NAS to host my personal website for 5
years, I finally decided to switch.
The main reason is that I evolved into a total git-geek---I want to
version control absolutely everything in git.
Wordpress' database approach for content management is overkill for my
purpose.
Instead, a static website versioned by git and hosted by GitHub would
meet all my needs.
